# Samples

This folder is for small, clearly redistributable samples that clarify how a format behaves.

Preferred sample types:

- ASCII layouts
- synthetic quarter maps
- simple generated diagrams
- public-domain or self-authored placeholder art

Avoid committing:

- proprietary sample packs
- commercial RPG Maker art
- any tileset image with unclear redistribution terms
